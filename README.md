# flybrainaligner

This repo collects the related software and pipelines developed for aligning fly brains.

## Install

Install Vaa3D in Linux with [Qt][]4.

with shell script:
```
cd vaa3d
./build.linux -B -m -j8
```
With cmake:
```
cd vaa3d
./build.cmake -h5j install
```
