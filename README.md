# flybrainaligner
flybrainaligner is developed for aligning fly brains

## Install

Install Vaa3D in Linux with [Qt][]4 and [HDF5][](>v1.8.14).

with shell script:
```
cd vaa3d
./build.linux -B -m -j8
```
With cmake:
```
cd vaa3d
./build.cmake -platform linux-x86_64 -h5j
```
