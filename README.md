# flybrainaligner
flybrainaligner is developed for aligning fly brains

## Install

Install Vaa3D in Linux with [Qt][]4.

Without h5j format support (recommended):
```
cd vaa3d
./build.linux -B -m -j8
```
With h5j support (if necessary):
```
cd vaa3d
./build.cmake -platform linux-x86_64 -h5j
```
